Linkdump No 7

May 31, 2024

linkdump

I collect a lot of links to great articles and software projects, and this is my attempt to sort through them and share them with the rest of you.

Articles

  • Surfing the (Human-Made) Internet
    Mike Grindle wrote a very nice article about the small, more personal internet that still exists outside of the usual places that are considered “the internet” these days. If you’re here, you know that already of course, but it’s still a great read and maybe something to forward along to people who are disillusioned with the current state of the net.

  • Uh-oh - Messenger pioneer ICQ will be shut down at the end of June
    The end of an era, though of course ICQ hasn’t been what it once was for at least 15 years by now. Still, as the first instant messenger I ever used it has a special place in my heart and I’m a little sad to see it go.

  • == ICQ Special ==
    ICQ may officially be dead soon, but there are still projects out there that try to keep it and other old messengers (AIM, MSN) alive. I’m just going to dump some links here without much comment, but if you’re as nostalgic about ICQ as I am and want to get it up and running again, get in touch with me and maybe we can figure something out together :)

    • https://web.icq.com/
      The (at the time of posting) still working ICQ site. If you accept the terms and conditions and then select “login with password” at the top left, you can still login with your old ICQ number and password, if you remember them ;)
    • https://nina.chat/
      Very interesting project with the goal of recreating AOL, ICQ, AIM, and Yahoo… currently in beta and only available for financial backers, but I’m seriously considering donating because I love the idea.
    • https://escargot.chat/
      A network to be used with old MSN clients.
    • https://runningman.network/
      A network to be used with old AIM clients.
    • https://github.com/ox/aim-oscar-server
      The server software running the runningman network.
    • https://github.com/namikiri/iserverd
      Old ICQ server reimplementation. Compiles on modern Linux, but doesn’t work right. Compiles and runs fine on old Linux versions though, I tested it on Ubuntu 4.10.
    • http://kicq.ru/
      A Russian ICQ server. They’re using iserverd.
